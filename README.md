# CRM Dashboard - Customers List

## Project Overview
This project is a responsive CRM Dashboard developed using HTML, CSS, and JavaScript. The dashboard provides a clean and modern interface for viewing customer information, statistics, and customer status details.
The implementation is based on the provided design requirements and focuses on creating a user-friendly and visually organized dashboard layout.


## Features
- Dashboard layout with sidebar navigation
- Customer statistics cards
- Customer management table
- Active and Inactive status indicators
- Search functionality
- Responsive design
- Clean and structured user interface

## Technologies Used
- HTML
- CSS
- JavaScript
